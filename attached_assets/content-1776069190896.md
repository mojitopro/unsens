# UNSENS - OpenCode Integration

[Permalink: UNSENS - OpenCode Integration](https://github.com/mojitopro/unsens#unsens---opencode-integration)

## Overview

[Permalink: Overview](https://github.com/mojitopro/unsens#overview)

This repository (`unsens`) has been modified to integrate with OpenCode's API system, allowing connection to various LLM providers through OpenCode.

## Features

[Permalink: Features](https://github.com/mojitopro/unsens#features)

### Multi-Provider Support

[Permalink: Multi-Provider Support](https://github.com/mojitopro/unsens#multi-provider-support)

- **Anthropic** (Claude): Uses `ANTHROPIC_API_KEY` and `ANTHROPIC_BASE_URL`
- **OpenAI** (GPT): Uses `OPENAI_API_KEY` and `OPENAI_BASE_URL`
- **Ollama** (Local): Uses local Ollama instance at `http://localhost:11434`
- **OpenCode**: Connect through OpenCode's provider system

### Environment Variables

[Permalink: Environment Variables](https://github.com/mojitopro/unsens#environment-variables)

| Variable | Description | Default |
| --- | --- | --- |
| `OPENAI_API_KEY` | OpenAI API key | - |
| `OPENAI_BASE_URL` | OpenAI-compatible API endpoint | `https://api.openai.com/v1` |
| `ANTHROPIC_API_KEY` | Anthropic API key | - |
| `ANTHROPIC_BASE_URL` | Anthropic API endpoint | `https://api.anthropic.com` |
| `OLLAMA_MODEL` | Ollama model name | `dolphin3:8b` |
| `AGENT_WORKSPACE` | Workspace directory | `/home/runner/agent-workspace` |
| `PORT` | API server port | `3000` |

### API Endpoints

[Permalink: API Endpoints](https://github.com/mojitopro/unsens#api-endpoints)

- `POST /api/chat` \- Chat with AI model (SSE streaming)
- `GET /api/models` \- List available models

### Usage with OpenCode

[Permalink: Usage with OpenCode](https://github.com/mojitopro/unsens#usage-with-opencode)

To use this API with OpenCode, configure in your `opencode.json`:

```
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "unsens": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "UNSENS API",
      "options": {
        "baseURL": "http://localhost:3000/api",
        "apiKey": "optional-key"
      },
      "models": {
        "claude-sonnet": {
          "name": "Claude Sonnet via UNSENS"
        }
      }
    }
  },
  "model": "unsens/claude-sonnet"
}
```

Or set environment variables:

```
export OPENAI_API_KEY=your-api-key
export OPENAI_BASE_URL=http://localhost:3000/api
opencode --model unsens/claude-sonnet
```

## Quick Start

[Permalink: Quick Start](https://github.com/mojitopro/unsens#quick-start)

### 1\. Install dependencies

[Permalink: 1. Install dependencies](https://github.com/mojitopro/unsens#1-install-dependencies)

```
cd /data/data/com.termux/files/home/unsens
pnpm install
```

### 2\. Configure API keys

[Permalink: 2. Configure API keys](https://github.com/mojitopro/unsens#2-configure-api-keys)

```
# Option 1: Anthropic (Claude)
export ANTHROPIC_API_KEY=sk-ant-...

# Option 2: OpenAI (GPT)
export OPENAI_API_KEY=sk-...

# Option 3: OpenCode proxy (e.g., OFOXAI)
export OPENAI_API_KEY=your-ofoxai-key
export OPENAI_BASE_URL=https://api.ofox.ai/v1
```

### 3\. Start the API server

[Permalink: 3. Start the API server](https://github.com/mojitopro/unsens#3-start-the-api-server)

```
cd artifacts/api-server
PORT=3000 pnpm start
```

### 4\. Test the API

[Permalink: 4. Test the API](https://github.com/mojitopro/unsens#4-test-the-api)

```
curl -X POST http://localhost:3000/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [{"role": "user", "content": "Hello!"}],
    "provider": "openai",
    "model": "gpt-4o"
  }'
```

## API Request Format

[Permalink: API Request Format](https://github.com/mojitopro/unsens#api-request-format)

### POST /api/chat

[Permalink: POST /api/chat](https://github.com/mojitopro/unsens#post-apichat)

```
{
  "messages": [\
    {"role": "user", "content": "Write a hello world program"}\
  ],
  "provider": "openai",  // "openai", "anthropic", "ollama", or "auto"
  "model": "gpt-4o",     // model name
  "apiKey": "...",       // optional: override API key
  "baseUrl": "..."       // optional: override base URL
}
```

## Termux/Android Setup

[Permalink: Termux/Android Setup](https://github.com/mojitopro/unsens#termuxandroid-setup)

Run the startup script:

```
chmod +x scripts/start-unsens-termux.sh
./scripts/start-unsens-termux.sh
```

Or manually:

```
export AGENT_WORKSPACE="/data/data/com.termux/files/home/unsens-workspace"
export OPENAI_API_KEY="your-key"
cd artifacts/api-server
PORT=3000 pnpm start
```
